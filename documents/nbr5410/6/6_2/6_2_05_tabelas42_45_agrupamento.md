# NBR 5410 — 6.2.5.5 Agrupamento de Circuitos

**Norma:** ABNT NBR 5410:2004
**Seção:** 6.2.5.5
**Tabelas:** 42, 43, 44, 45

---

## Princípios Gerais

### 6.2.5.5.1 — Quando Aplicar os Fatores

As ampacidades das Tabelas 36 a 39 são válidas para o número de condutores carregados indicado em cada coluna. Para linhas com **total de condutores superior**, aplicar os fatores de agrupamento das Tabelas 42 a 45.

### 6.2.5.5.2 — Exclusão de Condutores Levemente Carregados

Condutores com corrente de projeto **≤ 30%** de sua ampacidade corrigida podem ser **desconsiderados** para o cálculo do fator de agrupamento aplicável ao restante do grupo.

### 6.2.5.5.3 — Aplicação para Métodos A1, A2, B1, B2, C, D (Tabelas 36 e 37)
- Tabelas base válidas para 2 ou 3 condutores carregados.
- Para número maior: Tabela 42 (camada única) ou Tabela 43 (múltiplas camadas).
- Para subterrâneos: Tabela 44 (diretamente enterrados) ou Tabela 45 (eletrodutos enterrados).

### 6.2.5.5.4 — Aplicação para Métodos E, F, G (Tabelas 38 e 39)
- Tabelas base válidas para 2 ou 3 condutores carregados.
- Para número maior: Tabela 42 (feixe / camada única) ou Tabela 43 (múltiplas camadas).

### 6.2.5.5.5 — Condutores Não-Semelhantes

Os fatores das Tabelas 42 a 45 são válidos para grupos de condutores **semelhantes** (mesma temperatura de serviço e seções dentro de 3 valores normalizados sucessivos).

Para grupos não-semelhantes, usar:
- **a)** Cálculo caso a caso (ABNT NBR 11301); ou
- **b)** Fator simplificado: **F = 1 / √n**, onde n = número de circuitos ou cabos multipolares.

> **NOTAS:**
> 1. Para grupos com temperaturas de serviço diferentes, a ampacidade de todos os condutores deve ser baseada na **menor temperatura máxima** do grupo.
> 2. Condutores 100% carregados foram assumidos no cálculo dos fatores — se o carregamento for inferior, os fatores podem ser aumentados.

**Notas da Tabela 42:**
1. Aplicável a grupos homogêneos de cabos uniformemente carregados.
2. Quando a distância horizontal entre cabos adjacentes for > 2 × diâmetro externo, **não é necessário** fator de redução.
3. O número de circuitos refere-se à quantidade de grupos de 2 ou 3 condutores isolados / unipolares (um por fase) ou cabos multipolares.
4. Para agrupamento com bipolares e tripolares mistos: contar o total de cabos como número de circuitos; usar coluna de 2c para bipolares e 3c para tripolares.
5. Valores médios com dispersão geralmente < 5%.
