# NBR 5410 — 6.2.4 Seleção e Instalação em Função das Influências Externas

**Norma:** ABNT NBR 5410:2004
**Seção:** 6.2.4
**Tabela:** 34

> As prescrições relativas à seleção e instalação das linhas, sob o ponto de vista das influências externas indicadas em 4.2.6, são apresentadas na **Tabela 34** abaixo.

---

## Tabela 34 — Seleção e Instalação de Linhas em Função das Influências Externas

### A — Condições Ambientais (4.2.6.1)

#### AA — Temperatura Ambiente (4.2.6.1.1)

| Código | Faixa | Prescrição |
| :--- | :--- | :--- |
| AA1 a AA3 | Limite inferior < −10 °C | Condutores / cabos com isolação ou cobertura de PVC e condutos de PVC **não devem ser manipulados nem submetidos a esforços mecânicos** abaixo de −10 °C (PVC torna-se quebradiço) |
| AA4 a AA8 | Temperatura superior ao valor de referência (30 °C para instalações não-subterrâneas; 20 °C para enterradas) | As capacidades de condução devem ser reduzidas conforme **6.2.5.3.3** e **Tabela 40** |

#### AC — Altitude (4.2.6.1.3)
Sem influência sobre a seleção das linhas.

#### AD — Presença de Água (4.2.6.1.4)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AD1 | Desprezível | O uso de molduras em madeira só é permitido em AD1 |
| AD2 | Gotejamento | Cabos uni e multipolares com cobertura extrudada podem ser usados em qualquer tipo de linha, mesmo com condutos metálicos |
| AD3 a AD6 | Precipitação a jatos | Somente linhas com **proteção adicional à penetração de água** e graus IP adequados; em princípio sem revestimento metálico externo |
| AD7 | Imersão | Cabos uni e multipolares com **isolação resistente à água** (ex: EPR, XLPE) |
| AD8 | Submersão | **Cabos especiais para uso submerso** |

#### AE — Presença de Corpos Sólidos (4.2.6.1.5)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AE1 | Desprezível | Nenhuma limitação |
| AE2 | Pequenos objetos | Nenhuma limitação, desde que não haja exposição a danos mecânicos |
| AE3 | Objetos muito pequenos | Nenhuma limitação |
| AE4 a AE6 | Poeira leve a intensa | Precauções para evitar deposição que prejudique a dissipação térmica; escolher método que facilite a remoção da poeira |

#### AF — Presença de Substâncias Corrosivas ou Poluentes (4.2.6.1.6)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AF1 | Desprezível | Nenhuma limitação |
| AF2 | Atmosférica | Linhas protegidas contra corrosão / agentes químicos; cabos com cobertura extrudada são considerados adequados; condutores isolados só em eletrodutos com resistência adequada |
| AF3 | Intermitente | Idem AF2 |
| AF4 | Permanente | Somente **cabos uni ou multipolares adequados** aos agentes químicos presentes |

#### AG — Choques Mecânicos (4.2.6.1.7)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AG1 | Fracos | Nenhuma limitação |
| AG2 | Médios | Linhas com proteção leve; cabos usuais são adequados; condutores isolados em eletrodutos conforme ABNT NBR 5624 e 6150 |
| AG3 | Severos | Linhas com proteção reforçada; cabos com **armação metálica**; condutores isolados em eletrodutos conforme ABNT NBR 5597 e 5598 |

#### AH — Vibrações (4.2.6.1.7)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AH1 | Fracas | Nenhuma limitação |
| AH2 | Médias | Podem ser necessárias **linhas flexíveis** |
| AH3 | Severas | Somente **linhas flexíveis** (cabos uni ou multipolares flexíveis, ou condutores isolados flexíveis em eletroduto flexível) |

#### AK — Presença de Flora ou Mofo (4.2.6.1.8)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AK1 | Desprezível | Nenhuma limitação |
| AK2 | Prejudicial | Avaliar: cabos com armação (enterrados), condutores em condutos com IP adequado, materiais especiais ou revestimento em cabos / eletrodutos |

#### AL — Presença de Fauna (4.2.6.1.9)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AL1 | Desprezível | Nenhuma limitação |
| AL2 | Prejudicial (roedores, cupins) | **Cabos com armação**, ou condutores em condutos com IP adequado, ou materiais aditivados / revestimentos especiais |

#### AN — Radiação Solar (4.2.6.1.11)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| AN1 | Desprezível | Nenhuma limitação |
| AN2 | Média | Cabos ao ar livre ou em condutos abertos **resistentes às intempéries**; elevação de temperatura da superfície deve ser considerada no cálculo de ampacidade |
| AN3 | Alta | Idem AN2 |

### B — Utilização

#### BA — Competência das Pessoas (4.2.6.2.1)
Sem influência sobre a seleção das linhas.

#### BB — Resistência Elétrica do Corpo Humano (4.2.6.2.2)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| BB1 / BB2 | Alta / Normal | Nenhuma limitação |
| BB3 / BB4 | Baixa / Muito baixa | Ver 5.1 e seção 9 |

#### BC — Contato das Pessoas com o Potencial da Terra (4.2.6.2.3)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| BC1 | Nulo | Nenhuma limitação |
| BC2 / BC3 / BC4 | Raro a contínuo | Ver 5.1 e seção 9 |

#### BD — Fuga das Pessoas em Emergências (4.2.6.2.4)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| BD1 | Normal | Nenhuma limitação |
| BD2 / BD3 | Longa / Tumultuada | — |
| BD4 | Longa e tumultuada | Ver 5.2.2.2 |

#### BE — Natureza dos Materiais Processados ou Armazenados (4.2.6.2.5)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| BE1 | Riscos desprezíveis | Nenhuma limitação |
| BE2 | Riscos de incêndio | Ver 5.2.2.3 |
| BE3 | Riscos de explosão | Linhas protegidas por maneira de instalar adequada; ver ABNT NBR 9518 |
| BE4 | Riscos de contaminação | Linhas protegidas por maneira de instalar adequada |

### C — Construção das Edificações

#### CA — Materiais de Construção (4.2.6.3.1)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| CA1 | Não combustíveis | Nenhuma limitação |
| CA2 | Combustíveis | Ver 5.2.2.4 |

#### CB — Estrutura das Edificações (4.2.6.3.2)

| Código | Condição | Prescrição |
| :--- | :--- | :--- |
| CB1 | Riscos desprezíveis | Nenhuma limitação |
| CB2 | Sujeitas a propagação de incêndio | Ver 5.2.2.5 |
| CB3 | Sujeitas a movimentação | **Linhas flexíveis** ou contendo juntas de dilatação / expansão |
| CB4 | Flexíveis ou instáveis | **Linhas flexíveis** |
